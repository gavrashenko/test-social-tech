# test-social-tech  
  
## Project setup  
```sh  
git clone https://github.com/yurii-kozak/test-social-tech.git
cd test-social-tech
npm i
```  
  
## Using

### Step 1. Run Server Side
```  
npm run serve-server
```  
### Step 2. Run Client Side
```  
npm run serve-client
```  


  
#### Lints and fixes files  
```  
npm run lint  
```  
  
#### Run your unit tests  
```  
npm run test:unit  
```
